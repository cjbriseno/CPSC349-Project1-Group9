Project's Description

In this project, we built a website presenting a collection of photos about the sun ... .The project requires Node.js and NPM. We also used Eleventy static site generator to quickly generate sites with templates. Nu Html Check are used to verify the html codes.

The code written to be checked at local host is on main branch. The code published to Github Page is on deployPage branch.
